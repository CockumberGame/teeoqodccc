## 🎬 Анимации

### Появление карт
| Событие | Анимация | Длительность |
|---------|----------|--------------|
| Новый клиент | Падение сверху + shimmer | 2 сек |
| Новые карты действий | Выезд снизу | 2 сек |
| Использование карты | Тряска + исчезновение | 0.3 сек |
| Активация предмета | Покачивание | 0.5 сек |

### CSS Keyframes
```css
@keyframes slideDown {
  from { opacity: 0; transform: translateY(-100px); }
  to { opacity: 1; transform: translateY(0); }
}

@keyframes slideUp {
  from { opacity: 0; transform: translateY(100px); }
  to { opacity: 1; transform: translateY(0); }
}

@keyframes shimmer {
  0%, 100% { box-shadow: 0 0 10px rgba(0, 217, 255, 0.3); }
  50% { box-shadow: 0 0 25px rgba(0, 217, 255, 0.8); }
}

@keyframes shake {
  0%, 100% { transform: translateX(0); }
  25% { transform: translateX(-3px); }
  75% { transform: translateX(3px); }
}

@keyframes wobble {
  0%, 100% { transform: rotate(0deg); }

}
```

## 🎴 Компоненты карточек

### Двойная рамка (DOS стиль)
```css
.card-border-outer {
  border: 3px solid #e94560;
  border-radius: 8px;
  padding: 3px;
  background: #e94560;
  box-shadow: 0 4px 15px rgba(233, 69, 96, 0.4);
}

.card-border-inner {
  border: 2px solid #00d9ff;
  border-radius: 5px;
  background: #000000;
}
```

### Точечная шкала возбуждения
```
🍆 [████████░░] 60% ВОЗБУЖДЁН
```

**Логика:**
- 10 блоков total
- `█` = заполненный блок
- `░` = пустой блок
- Цвет меняется от процента:
  - 0-30%: `#00d9ff` (голубой)
  - 30-60%: `#00ff88` (зелёный)
  - 60-80%: `#ffaa00` (оранжевый)
  - 80-100%: `#ff0055` (красный)

## 🔧 Технические детали

### Фиксированные размеры
- **Карта клиента:** min-height: 280px
- **Карта действия:** min-height: 200px
- **Описание клиента:** max-height: 80px, overflow-y: auto

### Шрифты
- **Основной:** `Courier New`, Courier, monospace
- **Размеры:**
  - Заголовки: 12-16px
  - Основной текст: 10-11px
  - Вторичный текст: 8-9px
  - Теги: 8px

### Адаптивность
- `max-width: 400px` для контента хаба
- `max-width: 900px` для игрового экрана
- `flex-wrap: wrap` для кнопок
- `gap: 8-15px` для отступов
