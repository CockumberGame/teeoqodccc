# Контент для взрослых — Руководство

## 📁 Структура

```
src/data/content/
├── client_descriptions.json    # Описания клиентов (размеры, фетиши)
├── actions.json                # Тексты действий (орал, ласки и т.д.)
├── events.json                 # События (рейды, соседи и т.п.)
└── dialogs.json                # Диалоги и реплики
```

## ✏️ Как редактировать контент

### 1. Откройте нужный JSON-файл
Например: `src/data/content/client_descriptions.json`

### 2. Добавьте новый вариант
```json
{
  "size_descriptions": {
    "large": {
      "arousal_variants": [
        "Ваш текст здесь",
        "Ещё один вариант"
      ]
    }
  }
}
```

### 3. Сохраните и перезапустите игру
```bash
npm run dev
```

## 📝 Принципы написания

### Тон (7.5-8/10)
- **Чувственно, но не порнографично**
- **Намёки вместо прямых описаний**
- **Атмосфера, а не физиология**

### Примеры:

**Слишком откровенно (фильтры могут сработать):**
```
❌ "Толстый член, вены вздулись, пахнет тестостероном"
```

**Оптимально (атмосферно, но в рамках):**
```
✅ "Внушительный, напряжён — каждая вена намекает на страсть"
```

**Слишком мягко (теряется атмосфера):**
```
⚠️ "Клиент ожидает услугу"
```

## 🔧 Технические детали

### Плейсхолдеры в описаниях:
- `${client_description}` — базовое описание клиента
- `${visual_assessment}` — визуальная оценка размера (например, "16см")

### Фетиши (доступные ID):
```
rough, gentle, spit, eye_contact, deep, fast, slow, 
teasing, swallow, multiple, oral_focus, hand_use
```

### Интенсивность:
- `slow` — медленные, нежные действия
- `normal` — стандартный темп
- `intense` — интенсивные, грубые действия

## 🛡️ Безопасность контента

### Что может триггерить фильтры:
1. Прямые физиологические описания половых актов
2. Откровенные сцены с деталями
3. Определённые комбинации слов

### Как избежать:
1. Используйте метафоры и намёки
2. Фокусируйтесь на ощущениях, а не анатомии
3. Распределяйте «горячие» описания по разным файлам

## 📊 Баланс контента

Рекомендуемое соотношение:
- **60%** — атмосферные описания (намёки)
- **30%** — чувственные детали (эмоции, ощущения)
- **10%** — откровенные моменты (кульминация)

## 🔄 Обновление контента

Если хотите добавить новый тип контента:

1. Создайте файл `src/data/content/новый_контент.json`
2. Добавьте импорт в `ContentLoader.js`:
```javascript
import новыйКонтент from '../data/content/новый_контент.json' assert { type: 'json' };
```
3. Добавьте в конструктор:
```javascript
this.content = {
  descriptions: clientDescriptions,
  новыйКонтент: новыйКонтент
};
```
