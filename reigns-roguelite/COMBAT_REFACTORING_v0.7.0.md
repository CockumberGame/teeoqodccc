5. **Генерация 2 карт** с учётом фазы клиента

## 🎯 МАТРИЦА ДЕЙСТВИЙ (8×4)

### Категории (8):

| Категория | Ключ | Описание | Требование |
|-----------|------|----------|------------|
| **Ласки руками** | `hand` | Базовые ласки руками | Нет |
| **Ласки головой** | `head` | Ласки без проникновения | Нет |
| **Работа с членом** | `shaft` | С предметом (линейка/игрушка) | Предмет |
| **Оральный секс** | `bj` | Минет | Нет |
| **Глубокое горло** | `deep` | Глубокое проникновение | Оральный 5+ |
| **Дразнение** | `t_ease` | Сброс arousal, бонусы к деньгам | Наблюдательность 3+, Выносливость 2+ |
| **Доминантность** | `rough` | Доминирование над клиентом | Репутация 5+, Наблюдательность 5+, Внешность 5+ |
| **Особые** | `special` | Уникальные действия (босс-лут) | Предмет босса |

### Интенсивности (4):

| Интенсивность | Ключ | Стамина | Психика | Arousal | Успех |
|---------------|------|---------|---------|---------|-------|
| **Нежное** | `soft` | 3 | 0 | 8-14 | 90-95% |
| **Среднее** | `mid` | 5 | 1 | 15-22 | 80-85% |
| **Интенсивное** | `intense` | 8 | 2 | 25-32 | 65-70% |
| **Особое** | `special` | 12-20 | 4-8 | 35-60 | 40-60% |

### Примеры ключей действий:

```
hand_soft, hand_mid, hand_intense, hand_special
head_soft, head_mid, head_intense, head_special
shaft_soft, shaft_mid, shaft_intense, shaft_special
bj_soft, bj_mid, bj_intense, bj_special
deep_soft, deep_mid, deep_intense, deep_special
t_ease_soft, t_ease_mid, t_ease_intense, t_ease_special
rough_soft, rough_mid, rough_intense, rough_special
special_soft, special_mid, special_intense, special_special
```

**Итого:** 8 × 4 = **32 типа действий**

|------|---------|------|----------|-----------------|

### Флаг `gets_softer` (не фаза!)

**Триггер:** Неудачное действие с потерей arousal

**Эффекты:**
- 🧠 -5 психики игроку
- 😠 30% шанс ухода клиента
- 📉 -15 терпения
- 🎨 Визуально: серый фильтр на карте

**Восстановление:**
- Автоматически через 1 ход
- Или действие `t_ease_*`

## 📚 8 НАВЫКОВ

### Базовые (доступны с уровня 1):

| Навык | ID | Макс. уровень | Эффект |
|-------|-----|---------------|--------|
| **Выносливость** | `stamina_max` | 10 | +20 макс. стамины/ур, +AP на 5 и 10 уровне |
| **Наблюдательность** | `observation` | 10 | +10% шанс открыть фетиш/ур, +1 линейка осмотра |
| **Ловкость рук** | `hand_skill` | 10 | +5% успех hand/shaft карт, открывает mid/intense/special |
| **Оральный** | `oral_skill` | 10 | +5% успех bj карт, открывает mid/intense/special |

### Продвинутые (требования):

| Навык | ID | Требования | Эффект |
|-------|-----|------------|--------|
| **Глубокое горло** | `deep_skill` | Оральный 5+ | +5% успех deep карт |
| **Дразнение** | `teasing_skill` | Наблюдательность 3+, Выносливость 2+ | +5% деньги в edge, сброс arousal |
| **Внешность** | `appearance` | Динамически от репутации | +5% награда, +10% чаевые, +2% старт arousal |
| **Доминантность** | `dominance_skill` | Репутация 5+, Наблюдательность 5+, Внешность 5+ | +5% успех rough карт |

| Уровень навыка | Требуемая репутация | Бонус |

| 1-4 | 0+ | Базовый |

| 8-10 | 8+ | ✨ Очень красивая (+25% чаевые) |

## 🎮 ПРОКАЧКА НАВЫКОВ

### Система:

```
1 уровень репутации = 1 очко навыка
```

### Расход:

```
1 очко навыка = +1 уровень любого доступного навыка
```

### Проверка требований:

```javascript
// Пример: проверка Deep навыка
if (oral_skill.level >= 5) {
  deep_skill.unlocked = true;
}

// Пример: проверка Domination навыка
if (reputation.level >= 5 && 
    observation.level >= 5 && 
    appearance.level >= 5) {
  dominance_skill.unlocked = true;
}
```

## 🎴 ГЕНЕРАЦИЯ КАРТ (2 карты на ход)

### Алгоритм:

```
1. Определить текущую фазу клиента
2. Получить доступные игроку карты (по навыкам и предметам)
3. Выбрать 1 карту под текущую фазу (или соседнюю интенсивность)
4. Выбрать 1 случайную карту из доступных
5. Если нет карт под фазу → "одолжить" базовую карту
```

### Визуальная подсказка:

```
┌─────────────────────────┐ ┌─────────────────────────┐
│ [hand_soft]             │ │ [tease_intense]         │
│ ⚡3 🧠0 +8 arousal      │ │ ⚡6 🧠2 -20 arousal     │
│ Шанс: 95%               │ │ Шанс: 70%               │
└─────────────────────────┘ └─────────────────────────┘
```

**Примечание:** Нет пометок "РЕКОМЕНДУЕТСЯ" — игрок видит только статы

## 📊 ФОРМУЛЫ БОЯ

### Успех действия:

```
successChance = baseChance 
              + (skillLevel × 0.05) 
              + (appearance.level × 0.02)
              - phasePenalty 
              - diminishingReturns

Кап: 10% ≤ successChance ≤ 95%
```

### Награда:

```
reward = baseReward (30)
       × sizeMultiplier
       × phaseMultiplier
       × (1 + appearanceBonus + edgeBonus)
       × satisfaction
```

### Arousal gain:

```
arousalGain = baseArousal 
            × phaseMultiplier 
            × (1 + skillBonus) 
            × diminishingReturns
```

### Diminishing returns:

| Повтор | Множитель |
|--------|-----------|
| Первое | 100% |
| Второе подряд | 75% |
| Третье подряд | 50% |
| 4+ подряд | 25% |
