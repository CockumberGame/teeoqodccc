## 🔴 Критичные проблемы

### **Проблема 1: Hub не открывается**

**Симптомы:**
- После ввода имени экран не меняется
- Консоль пустая
- phase остаётся 'menu'

**Возможная причина:**
```javascript
// В GameContext.jsx, useEffect инициализации:
if (saveData.success && engine.player) {
  dispatch({ type: 'INIT', ... });
}
```
`engine.player` может быть null после загрузки

**Решение:**
```javascript
// Проверить:
console.log('LoadGame result:', saveData);
console.log('Engine player:', engine.player);

// Исправить:
if (saveData.success) {
  dispatch({
    type: 'INIT',
    payload: { phase: 'hub', ... }
  });
}
```

---

### **Проблема 2: startNight не работает**

**Симптомы:**
- Кнопка "Начать ночь" не реагирует
- Ошибок в консоли нет

**Возможная причина:**
```javascript
// В HubScreen.jsx:
const { startNight } = useGame();
```
`startNight` может быть undefined

**Решение:**
```javascript
// Проверить в GameContext.jsx:
const value = {
  state,
  engine,
  startNight,  ← Есть ли в value?
  ...
};
```

---

### **Проблема 3: GameScreen не рендерится**

**Симптомы:**
- phase = 'playing' но экран пустой
- actionState = null

**Возможная причина:**
```javascript
// В GameScreen.jsx:
const { action } = state;
action?.currentActionCards?.length > 0
```
`action` может быть null

**Решение:**
```javascript
// Добавить проверку:
if (!action || !game) {
  return <div>Loading...</div>;
}
```
