## 🎯 ЖАНР И КОНЦЕПЦИЯ

### Это НЕ карточная игра в классическом понимании

**Gloryhole Quest** — это **RPG-бой с клиентами** в формате текстовой игры:

| Элемент | Аналогия в RPG | Реализация |
|---------|----------------|------------|
| **Клиент** | Враг / Босс | Имеет "HP" (arousal), фазы, черты, фетиши |
| **Arousal** | HP врага | Шкала 0-100%, заполняется действиями |
| **Карты действий** | Атаки / Способности | 2 карты на ход, выбор тактики |
| **Фазы arousal** | Стадии боя | Спокоен → Заинтересован → Возбуждён → Готов → На грани |
| **Стамина/Психика** | Ресурсы героя | Тратятся на действия, восстанавливаются в хабе |
| **Завершить раньше** | Критический удар | Попытка завершить бой досрочно |
| **Сбежать** | Отступление | Возврат в хаб со штрафом |


**6 фаз (v0.7):**
| Фаза | Arousal | Цвет | Описание | Доступные карты |
|------|---------|------|----------|-----------------|
| **soft** | -15–0% | 🩶 | Холодный | Только soft |
| **interested** | 0–25% | 🟢 | Заинтересован | soft + mid |
| **excited** | 25–50% | 🔵 | Возбуждён | soft + mid + intense |
| **ready** | 50–75% | 🟣 | Готов | Все карты |
| **edge** | 75–95% | 🟠 | На грани | Все + edge stacking |
| **climax** | 95–100% | 🔴 | Финал | Выбор исхода |

**Флаг gets_softer:** Неудачное действие → arousal падает, -5 психики игроку, 30% шанс ухода клиента.

### Стратегия

**Варианты:**
1. **Быстро:** Интенсивные карты → высокий риск раннего финала
2. **Медленно:** Дразнение → сброс arousal → бонус к деньгам (1.5x)
3. **Баланс:** Чередование фаз → макс. прибыль

## 🃏 КАРТОЧНАЯ МЕХАНИКА

### Колода действий

**НЕ классическая колода** (нет руки, сброса, добора)

**Генератор карт на каждый ход:**
- 2 карты из пула
- Учитывается фаза arousal
- Учитываются навыки игрока
- Учитываются открытые фетиши клиента

**Пул карт (~20 штук):**
- **Slow:** gentle_touch, light_suck, tease_tip, inspect
- **Normal:** normal_rhythm, eye_contact, verbal_play, hand_work
- **Intense:** deep_throat, fast_pace, rough_play, intense_suction

### Diminishing Returns

```
tease_tip (1x): 95% шанс, +14 arousal
tease_tip (2x): 80% шанс (-15%), +14 arousal
tease_tip (3x): 65% шанс (-30%), +14 arousal
tease_tip (4x): 🚫 ЗАБЛОКИРОВАНО
```

**Сброс:** При смене клиента счётчик обнуляется

## 🔥 EDGE STACKING (v0.8)

**Концепт:** Игрок может **намеренно сбрасывать arousal** для накопления бонусов к награде.

### Механика:

```
1. Достигни edge фазы (75-95% arousal)
2. Используй t_ease действие:
   - arousal: 75% → 70% (сброс)
   - edgeStacks: +1/+2/+3/+5 (от интенсивности)
3. Повтори для накопления стеков (макс 5)
4. Заверши клиента → получи бонус +10% за стек
```

### Баланс:

| Действие | arousal | edgeStacks | stamina | Условие |
|----------|---------|------------|---------|---------|
| `t_ease_soft` | -10 | +1 | 3 | arousal ≥ 75% |
| `t_ease_mid` | -15 | +2 | 5 | arousal ≥ 75% |
| `t_ease_intense` | -18 | +3 | 8 | arousal ≥ 75% |
| `t_ease_special` | -20 | +5 | 12 | arousal ≥ 75% |

### Пример:

```
Ход 1-3: Обычные действия → arousal 95% (edge фаза)
Ход 4: t_ease_intense → arousal 70%, edgeStacks: 3
Ход 5-6: Снова raising arousal → 95%
Ход 7: t_ease_soft → arousal 70%, edgeStacks: 4 (макс 5)
Ход 8-10: Финал → climax с +40% к награде

Без edge stacking: 100 money
С edge stacking: 140 money (+40 за 4 стека)

Риск: 3-4 дополнительных хода = больше шансов на неудачу
```
